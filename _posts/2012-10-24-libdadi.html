What is libdadi ?

libdadi has been developed to federate all the mandatory elements of every distributed and low-coupled applications.

libdadi currently implements the following functionalities:

  • command-line parsing
  • various types of logging (local and distributed)
  • a cross-platforms plugin mechanism
  • a attributes map allowing easy manipulation of various exportation file format such as XML, JSON or INI (limited)
  • an exception base classes structure
  • CoRI base implementation : system for metric gathering based on the plugin system and the attributes elements
