Here is the corresponding changelog:

  • New features :
    • Add PBS_Pro support
    • Add specific parameters options to batch schedulers (option -S for vishnu_submit_job)
    • Default parameters for a batch can be given throught a config file
    • New feature not functionnal for web services
  • Bug fixes and improvements :
    • Improve the generation of the ID for vishnu objects
    • Add documentation for admin with distant databases
    • Clean unused files
    • Fixed bug 440
    • Fix LSF bug for -N and -P options in submit
    • Fix bad types in comparison from unsigned int to size_t

You can download the tarball containing the sources on the dedicated Vishnu download Page or directly from the following link : Vishnu 2.0.0 beta3
