History

V2.0.0 Beta-3 (09/11/2012)
PBS_pro support added, specific and default parameters for batch schedulers scripts added (through options and config file resp.). Generation of the ID for vishnu objects improved. Several bug fixes and the documentation was improved
V2.0.0 Beta-2 (02/10/2012)
SGE batch support, extension of jobs submission options, fixed bugs 369, 399, 410, 417, 426, 431, 434.
V1.3.0 (30/03/2012)
LDAP support for authentication, LSF batch support, fixed bugs 335, 343.
V1.2.7 (19/03/2012)
Fix bugs (337, 326), extend TMS options, TMS enhancement (autosubmit, list all as machine id, ...).
V1.2.6 (11/01/2012)
Fix bugs, add options, minor API changes.
V1.2.5 (07/11/2011):
Fix bugs.
V1.2.4 (31/08/2011)
Change python API for connect/reconnect, fix typo problems.
V1.2.3 (23/08/2011)
Fix some bugs and add SLURM support.
V1.2.2 (21/07/2011)
Fix some bugs.
V1.2.1 (24/06/2011)
Fix problems with DNS resolution of VISHNU hosts.
V1.2.0 (15/06/2011)
Release of modules IMS (information mgmt services) and FMS (files mgmt services).
V1.1.0 (13/05/2011)
Release of module TMS (tasks mgmt services).
V1.0.1 (17/03/2011)
Release of first module UMS (users mgmt services).
