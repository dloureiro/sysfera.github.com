Installation

Requirements

libdadi can be installed on almost every system (if you find one we don't support, please provide patches :), and requires the following software:

  • CMake (2.8.4 or greater)
  • Boost C++ libraries (1.46.1 or greater)
  • Sigar library (1.6.2 or greater)

Installation procedure

libdadi is using the CMake build system.

Here is the list of commands to run in order to compile and install libdadi form the main directory:

{% highlight bash %} $ mkdir build $ mkdir target $ cd build $ ccmake .. -DCMAKE_INSTALL_PREFIX=$(pwd)/../target -DSIGAR_LIBRARIES=path_to_sigar_library -DSIGAR_INCLUDE_DIR=path_to_sigar_include_dir $ make && make install {% endhighlight %}
